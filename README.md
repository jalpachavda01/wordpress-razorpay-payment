# Razorpay Donation Integration (Custom)

This project demonstrates a custom Razorpay payment integration without using any plugin.

## 🔗 Live Demo
https://thecarouselofdreams.org/donate/

## 🚀 Features
- Custom donation button
- Razorpay checkout integration
- Success / failure handling
- Secure test mode implementation

## 🛠️ Technologies Used
- WordPress / PHP
- JavaScript
- Razorpay API
- HTML / CSS

## ⚠️ Security Note
All sensitive keys are removed. Only test keys are used in demo.
